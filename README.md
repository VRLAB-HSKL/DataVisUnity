# DataVisUnity
## Klonen
Bei dem Klonen dieses Repositories muss in Unity anschließend das ViveInputUtility-Toolkit aus dem Asset Store heruntergeladen werden. 
Des Weiteren kann es sein, dass bestimmte Prefabs und UI-Elemente nicht mehr gefunden werden. 
Tritt dieser Fall auf, müssen die UI-Elemente des Whiteboards erneut in die Felder des WhiteboardValueLoader Skripts hineingezogen werden. 
Anschließend müssen die Felder des auf dem ConfuisonMatrixCanvas befindlichen UIConfusionMatrixGenerator Skripts erneut mit den vorgesehenen Prefabs befüllt werden.
Diese Prefabs sind unter folgendem Pfad zu finden: 
- Assets/Resources/Prefabs
